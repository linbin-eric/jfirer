JVM中的方法分派

[TOC]

方法分派指的是虚拟机将代码解析为虚拟机运行中方法的内存入口地址的过程。Java 语言中存在两个分派的过程：静态分派和动态分派。

静态分配就是编译器期间完成的方法分派。可以根据多于一个的参数（方法接收者，以及方法入参）进行分派，分派参数的类型使用的变量的类型。

动态分派仅仅只使用一个参数，也就是实际的方法接收者类型来进行分派。

上述流程是一个概念模型，由于动态分派在运行过程中十分频繁，虚拟机会使用各种方式来加速查找流程。

总结

对于 Java 而言，其是一个静态多分派，动态单分派的语言。
